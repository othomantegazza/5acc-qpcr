## 5acc-qpcr

qPCR analysis for https://github.com/TomHarrop/ird-5acc-paper

### Requirements

All software is containerised, so the only requirements are:

- [`singularity`](https://singularity.lbl.gov) ≥ 2.4.1   
- [`snakemake`](https://snakemake.readthedocs.io) ≥ 4.7.0

### Reproduce the analysis

`snakemake --use-singularity`
